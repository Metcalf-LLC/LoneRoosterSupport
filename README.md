# LoneRoosterSupport

Public support site for LoneRooster account deletion and support links.

## Publish with GitHub Pages

1. Create a new **public** GitHub repo named `LoneRoosterSupport`.
2. Upload the contents of this folder to the root of that repo.
3. In the new repo, go to `Settings > Pages`.
4. Under `Build and deployment`, choose:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/docs`
5. Save.

Your account deletion URL will be:

`https://metcalf-llc.github.io/LoneRoosterSupport/account-deletion/`
